Visit project

Design & developed in 2024

Le hijab & le porno

Variables is a typography project using variable fonts to animate excerpts from Olivier Marboeuf’s poems. By adjusting font weight, width, and slant dynamically, the reading experience changes with each line. This group project combined a shared page with excerpts and individual pages showcasing each student’s unique artistic interpretation.
